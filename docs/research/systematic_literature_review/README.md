# DCLO PRISMA-Aligned SLR Package

This folder contains a rapid v0.1 systematic literature review package for the PhD project **Digital Capabilities for Life Outcomes**.

## Files

- `SLR_PRISMA_DCLO.md` - main narrative report and protocol.
- `SLR_PRISMA_DCLO.docx` - professor-facing Word version.
- `DCLO_TRANSLATIONAL_VENTURE_NOTE.md` - scoped pathway for a separate DCLO research/advisory practice.
- `SLR_SCOPUS_SCREENING_WORKBOOK.xlsx` - reviewer-ready Excel workbook for the 566-record Scopus screening queue.
- `search_log.csv` - PRISMA-S style search/audit log.
- `included_seed_records.csv` - initial evidence map and verification statuses.
- `screening_template.csv` - title/abstract and full-text screening template.
- `data_extraction_template.csv` - extraction fields for final SLR.
- `citation_verification_log.csv` - source verification queue.
- `scopus_screening_queue.csv` - all 566 Scopus records normalized and sorted for title/abstract screening.
- `scopus_priority_records.csv` - 260 P1/P2 Scopus records for first-pass human screening.
- `search_provenance_log.md` - source-discovery and search provenance log.
- `prisma_flow_counts.md` - pilot flow counts; replace with database counts for final SLR.
- `survey_instrument/` - Qualtrics survey blueprint, build script, codebook, flow logic, and ethics/recruitment note for primary data collection across all three papers.
- `rigour_pack/` - preregistration-style protocol, database query templates, PRISMA/PRISMA-S checklists, quality appraisal rubrics, dedupe protocol, computational reproducibility note, and reproducibility manifest.
- `raw_exports/scopus_dclo_core_566_2026-05-28.csv` - raw Scopus export from JGU academic-access session.

## Important status note

This is a PRISMA-aligned rapid protocol plus evidence map. It is suitable for a supervisor meeting and for launching the final SLR. It should not be represented as a completed publishable PRISMA systematic review until database exports, dedupe, dual screening, full-text appraisal, and final PRISMA flow are complete.
